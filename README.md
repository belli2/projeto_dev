# Software da Web para Abertura de Ordem de Serviços de Manutenção
## Descrição:
```
O software da web permitirá que os funcionários de uma empresa abram ordens de serviço de manutenção, detalhando a natureza do problema ou necessidade. Os funcionários da manutenção poderão então fazer a triagem dessas ordens de serviço, priorizando e atribuindo tarefas conforme necessário. Após atender ao chamado, a manutenção poder á encerrar a ordem de serviço, inserindo uma foto como comprovação do trabalho realizado. Também deverá permitir o acompanhamento do status de cada ordem de serviço, desde a abertura até o fechamento, garantindo uma gestão eficiente das solicitações de manutenção.

```
## Tecnologias Utilizadas: 
```
* Phyton (backend)
* JavaScript (backend)
* NODE.JS (bibliotecas)
* ReactNative
* HeidiSQL (banco de dados)
* GitHub (orgnização e interação)
* VS Code (programação)

```
