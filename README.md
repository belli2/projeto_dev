# Software da Web para Abertura de Ordem de Serviços de Manutenção
## Descrição:
```
O software da web permitirá que os funcionários de uma empresa abram ordens de serviço de manutenção, detalhando a natureza do problema ou necessidade. Os funcionários da manutenção poderão então fazer a triagem dessas ordens de serviço, priorizando e atribuindo tarefas conforme necessário. Após atender ao chamado, a manutenção poderá encerrar a ordem de serviço, inserindo uma foto como comprovação do trabalho realizado. Também deverá permitir o acompanhamento do status de cada ordem de serviço, desde a abertura até o fechamento, garantindo uma gestão eficiente das solicitações de manutenção.

```
## O problema
```
Atualmente, a forma como  a solicitação da manutenção é feita por WhatsApp, o que leva a uma informalidade  e nem sempre uma visualização de todas as tarefas.

```
## Público-alvo
```
* Professores
* Coordenadores
* Funcionários da manutenção

```
## Objetivos
```
* Facilitar a ordem de prioridade da manutenção
* Ter um contato mais organizado entre as partes
* Ter a certeza da conclusão da tarefa

```
